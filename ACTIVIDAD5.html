Descripción del Proyecto

Objetivo: Utilizar una única función para encender y apagar el LED y la alarma según la distancia medida.

Funcionamiento: Encender un foco si la distancia es mayor a 100cm, si es menor encender una alarma.

Simulación tinkercad

Haz clic en "Iniciar simulación" y prueba moviendo el objeto virtual cerca y lejos del sensor

Componentes Utilizados

Arduino Uno

Microcontrolador principal que procesa las señales y controla los actuadores

Sensor HC-SR04

Sensor ultrasónico que mide distancias de 2cm a 400cm usando ondas sonoras

LED

Indicador luminoso que se enciende cuando el objeto está lejos (>100cm)

Buzzer

Alarma sonora que se activa cuando el objeto está cerca (≤100cm)

Código Arduino

Función Principal del Código

El código utiliza una sola función llamada controlarSistema() que:

  • Recibe la distancia medida como parámetro
  • Evalúa si es mayor o menor a 100cm
  • Controla LED y buzzer según la condición
  • Muestra el estado en el monitor serie

Cómo Funciona

1️. Medición de Distancia

El sensor HC-SR04 envía ondas ultrasónicas que rebotan en los objetos. El tiempo que tardan en regresar se convierte en distancia usando la fórmula de velocidad del sonido.

2️. Función de Control

La magia ocurre en controlarSistema()

  • Si distancia > 100cm: LED ON, Buzzer OFF
  • Si distancia ≤ 100cm: LED OFF, Buzzer ON

Una sola función maneja ambos dispositivos de forma eficiente

3️. Monitoreo en Tiempo Real

El monitor serie muestra las lecturas de distancia y el estado actual del sistema, facilitando la depuración y verificación.

Instrucciones de Uso

  1. Abre Tinkercad: Ve al enlace de la simulación arriba
  2. Inicia la simulación: Haz clic en "Iniciar simulación"
  3. Prueba el sensor: Mueve el círculo azul (objeto) cerca y lejos del sensor
  4. Observa: LED se enciende cuando está lejos, alarma cuando está cerca
  5. Monitor serie: Ve las lecturas en tiempo real

Resultados Esperados

El sistema funcionará así:

  • Objeto lejos (>100cm): LED encendido, sin alarma
  • Objeto cerca (≤100cm): Alarma sonando, LED apagado
  • Monitor serie: Mostrará distancias y estados
  • Respuesta: Cambios instantáneos cada 0.5 segundos
