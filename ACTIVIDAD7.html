ACTIVIDAD 7

Sensor de Distancia con Display LCD

Objetivo: Crear un sistema que mida distancias, las muestre en un LCD y controle actuadores según la proximidad.

Requisitos del ejercicio:

  • Primera fila LCD: mostrar la palabra "DISTANCIA"
  • Segunda fila LCD: mostrar el valor de distancia + "cm"
  • Control automático: Si distancia > 100cm → encender foco
  • Control automático: Si distancia ≤ 100cm → encender alarma

Componentes del Sistema

Arduino Uno

Cerebro del sistema que procesa sensores y controla actuadores

Sensor HC-SR04

Mide distancias de 2-400cm usando ultrasonido

LCD 16x2

Pantalla que muestra "DISTANCIA" y el valor medido

LED

Se enciende cuando la distancia es mayor a 100cm

Buzzer

Alarma que suena cuando la distancia es ≤ 100cm

Potenciómetro

Controla el contraste del LCD para mejor visibilidad

Código Arduino Completo

Características Principales del Código

  • Display LCD: Muestra "DISTANCIA" en fila 1 y valor+unidad en fila 2
  • Medición continua: Actualiza distancia cada 500ms
  • Control automático: Activa LED o alarma según umbral de 100cm
  • Actualización en tiempo real: LCD se actualiza constantemente

Funcionamiento del Sistema

1️. Lectura del Sensor

El sensor HC-SR04 envía pulsos ultrasónicos que rebotan en los objetos. El tiempo de ida y vuelta se convierte a distancia usando la velocidad del sonido.

2️. Actualización del LCD

Formato del Display:

  • Fila 1: "DISTANCIA" (centrado y fijo)
  • Fila 2: "[valor] cm" (centrado y dinámico)
  • Actualización: Cada 500ms
  • Precisión: 1 decimal (ej: "45.3 cm")

3️.Control Automático

Lógica de Control:

Distancia > 100cm:
  • LED: ENCENDIDO
  • Alarma: APAGADA
  • Estado: "Objeto lejano"
Distancia ≤ 100cm:
  • LED: APAGADO
  • Alarma: ENCENDIDA
  • Estado: "Objeto cerca"

Instrucciones de Uso

  1. Conexión: Verifica que todos los componentes están bien conectados
  2. Contraste: Ajusta el potenciómetro para ver bien el LCD
  3. Subir código: Carga el programa al Arduino
  4. Inicialización: Espera 2 segundos a que aparezca "DISTANCIA"
  5. Prueba: Mueve objetos cerca y lejos del sensor
  6. Observa: Ver cambios en LCD, LED y alarma en tiempo real

Resultados Esperados

El sistema completo funcionará así:

  • LCD siempre muestra: "DISTANCIA" en fila 1, valor+cm en fila 2
  • Medición continua: Actualización cada medio segundo
  • LED automático: Se enciende solo cuando distancia > 100cm
  • Alarma automática: Suena solo cuando distancia ≤ 100cm
  • Monitor serie: Información detallada para debugging
  • Precisión: Mediciones confiables de 2cm a 400cm

Características Avanzadas

Monitoreo Serie

El código incluye información detallada en el monitor serie para facilitar el debugging y seguimiento del sistema.

Centrado Automático

El texto en el LCD se centra automáticamente calculando los espacios necesarios según la longitud del valor.

Optimizaciones

  • Limpieza del LCD en cada actualización para evitar caracteres residuales
  • Limitación de valores extremos del sensor
  • Inicialización con mensaje de bienvenida
  • Código modular con funciones específicas

Posibles Mejoras

  • Agregar promediado de lecturas para mayor estabilidad
  • Incluir múltiples umbrales con diferentes acciones
  • Añadir LEDs de colores para diferentes rangos
  • Implementar alarma con diferentes tonos
  • Guardar valores máximos y mínimos detectados
  • Crear interfaz con botones para configurar umbral
